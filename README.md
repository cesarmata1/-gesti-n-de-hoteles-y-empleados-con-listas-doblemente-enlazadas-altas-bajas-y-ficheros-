# Gestión de Hoteles y Empleados (Listas Doblemente Enlazadas)

Este repositorio contiene un proyecto desarrollado en **C++** para gestionar información de hoteles y empleados utilizando **listas doblemente enlazadas** y manejo de ficheros. Es una práctica que forma parte de las actividades realizadas en clase.

## Descripción

El código implementa un sistema sencillo de gestión que permite:
- Añadir hoteles y empleados.
- Eliminar registros.
- Guardar y leer datos desde ficheros para mantener la información entre sesiones.

El objetivo de la práctica es aplicar estructuras de datos complejas (listas doblemente enlazadas) junto con operaciones de entrada/salida de ficheros en C++.

## Tecnologías utilizadas

- C++
- Compilador compatible con C++ (por ejemplo, g++)
